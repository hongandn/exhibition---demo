DNB Exhibition Catalogue Data Model

Linked Open Exhibition

A unified data model for exhibition catalogues aligned with CIDOC CRM, Wikibase4Research, and Wikidata standards
Published

July 17, 2026

Overview

This data model provides a standardized schema for describing exhibition catalogues and bibliographic information within the Linked Open Exhibition ecosystem. It is aligned with three major sector standards:

  • CIDOC CRM (Conceptual Reference Model) v7.1.3
  • Wikibase4Research (WB4R) - NFDI4Culture platform
  • Wikidata - WikiProject Books

Best Practices for Catalogue Records

  1. Use Wikidata Items: Always link to existing Wikidata items for authors (P50), publishers (P123), and organizations to enable cross-institutional discovery.

  2. ISBN Validation: Ensure ISBN-13 numbers are valid according to the standard checksum algorithm. Use the format with hyphens (978-X-XXXX-XXXX-X) for readability.

  3. GND Integration: Cross-reference GND identifiers from the German National Library to enhance discoverability within German cultural heritage networks.

  4. Publication Dates: Use ISO 8601 format (YYYY-MM-DD) where precise dates are available, or just YYYY for year-only precision.

  5. Multilingual Titles: Where catalogues are published in multiple languages, create additional title entries using monolingual text or language-qualified items.

  6. Cover Images: Upload catalogue cover images to Wikimedia Commons (P18) to provide visual context and enhance accessibility.

  7. URL Persistence: Prefer institutional persistent URLs (with DOI or similar) over temporary web links when available to ensure long-term reference stability.
