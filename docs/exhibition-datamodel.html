Exhibition Data Model

Linked Open Exhibition

A unified data model for exhibitions aligned with CIDOC CRM, Wikibase4Research, and Wikidata standards
Author

Hong An Dinh

Published

July 20, 2026

Overview

This data model provides a standardized schema for describing exhibitions and their contextual information within the Linked Open Exhibition ecosystem. It is aligned with three major sector standards:

  • CIDOC CRM (Conceptual Reference Model) v7.1.3
  • Wikibase4Research (WB4R) - NFDI4Culture platform
  • Wikidata - WikiProject Exhibitions

Best Practices for Exhibition Records

  1. Use QIDs for References: Always link to existing Wikidata items when available (Q464980 for exhibitions, institution Q-numbers, country Q-numbers, etc.)

  2. Date Formatting: Use ISO 8601 format (YYYY-MM-DD) for all temporal data. For exhibitions spanning years, precision can be reduced to YYYY (year only).

  3. Multilingual Titles: Where applicable, add exhibition titles in multiple languages using separate items or monolingual text fields.

  4. Institutional Context: Link to the organizing institution’s Wikidata item (P17 for country, P664 for organizer) to enable cross-institutional exhibition discovery.

  5. Participant Documentation: Ensure all participating artists have Wikidata items. Create items for lesser-known artists to enrich the knowledge graph.

  6. Commons Integration: Regularly link exhibition items to Wikimedia Commons categories (P373) to support multimedia documentation and community contributions.
